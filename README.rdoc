= BR::CPF

Lib implementada em C (gera uma lib compilada nativamente) que calcula e valida CPFs.

== Instalação

 sudo gem install br-cpf

== Exemplos

=== Usar
 require 'rubygems'
 require 'br/cpf'

=== Validar

 BR::CPF.valid? '00000000191'
  => true

 BR::CPF.valid? 12345678909
  => true

 BR::CPF.valid? 12345678900
  => false

=== Gerar CPF

 raiz = 1
 BR::CPF.new(raiz).to_s
  => '00000000191'
